# task-app
# タスク管理アプリ

## 概要

Reactで作成した、業務利用を想定したタスク管理アプリです。

単なるToDoアプリではなく、**管理画面でよく使われる基本機能**を意識して実装しました。

一覧表示、検索、絞り込み、追加・編集・削除、入力バリデーション、localStorageによるデータ永続化に対応しています。

---

## 制作目的

フロントエンド副業案件では、見た目だけでなく以下のような実装力が求められると考え、このアプリを作成しました。

* フォーム実装
* CRUD操作（追加 / 編集 / 削除）
* 一覧表示
* 検索・絞り込み
* 状態管理
* 永続化処理

実務でよくある「業務管理画面」の基礎を、Reactで自力実装することを目的にしています。

---

## 主な機能

### 1. タスク追加

* タイトル
* 説明
* ステータス
* 優先度
* 期限

を入力してタスクを登録できます。

### 2. タスク編集

既存タスクの内容を編集できます。

### 3. タスク削除

不要なタスクを削除できます。

### 4. 完了切り替え

タスクをワンクリックで完了 / 未完了に切り替えできます。

### 5. 検索機能

タイトル・説明文に対してキーワード検索ができます。

### 6. 絞り込み機能

* ステータス別
* 優先度別

で絞り込みができます。

### 7. データ永続化

localStorageを用いて、ページ再読み込み後もデータが保持されます。

### 8. バリデーション

* タイトル必須
* タイトル50文字以内
* 説明300文字以内

の入力制御を実装しています。

## 工夫したポイント

### 業務画面を意識したUI構成

単純なToDoリストではなく、案件管理や顧客管理画面にも流用しやすいレイアウトを意識しました。

### 実務で使う基本操作を一通り実装

React案件でよく求められる、以下の基本要素を一つのアプリにまとめています。

* CRUD
* モーダルUI
* フォーム入力
* バリデーション
* 一覧 / 検索 / 絞り込み
* ステータス管理

### 状態管理をシンプルに整理

useState / useEffect / useMemo を使い、シンプルながら読みやすい構成を意識しました。
